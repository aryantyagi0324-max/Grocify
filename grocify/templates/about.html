Core Features

Inventory Management

Track food items with expiry dates, quantities, and categories. Full CRUD operations with search and filtering.

Expiry Tracking

Automated expiry detection with color-coded alerts. Reduce food waste with timely notifications.

Recipe Suggestions

Real-time recipe suggestions using TheMealDB API. Cooking instructions, ingredients, and YouTube links.

Technology Stack

Django 4.x

Backend framework with authentication and ORM

JavaScript

Frontend interactivity and API calls

SQLite

Database with Django ORM management

TheMealDB API

Real recipe data integration

System Architecture

Full-Stack Django Application - Follows MVC pattern with clean separation of concerns

Architecture Layers

  • Presentation Layer: HTML templates with CSS/JS
  • Business Logic: Django views and models
  • Data Layer: SQLite database with ORM
  • External APIs: TheMealDB for recipe data

Key Features

  • User authentication system
  • Search and filtering
  • Dark/Light theme toggle
  • Responsive design
